# Adidas Sales Analysis Dashboard

## Overview

This project analyzes Adidas sales data using Microsoft Excel to uncover insights into sales performance, operating profit, retailers, product categories, and sales methods. The analysis was completed using Pivot Tables, Pivot Charts, and an interactive dashboard to help visualize key business metrics.

## Project Objectives

- Analyze overall sales performance.
- Compare operating profit across retailers.
- Identify the best-performing product categories.
- Evaluate sales by different sales methods.
- Build an interactive dashboard for business decision-making.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Cleaning
- Data Analysis

## Dashboard Features

The dashboard includes:

- Retailer Performance Analysis
- Product Category Analysis
- Sales Method Analysis
- Total Sales Comparison
- Operating Profit Comparison
- Interactive Pivot Filters

## Key Insights

- **West Gear** recorded the highest total sales among all retailers.
- **Foot Locker** was another top-performing retailer.
- **Men's Street Footwear** generated the highest sales among product categories.
- **In-store** sales contributed the largest share of total sales compared to Online and Outlet sales methods.
- Operating profit generally followed the same trend as total sales across retailers and product categories.
